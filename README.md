# Plise Perde Sistemleri

Türkiye üretimi yüksek kaliteli plise perde sistemleri. Blackout, honeycomb, gece gündüz ve özel ölçü pliseli perde modelleri.

## Özellikler

- Blackout plise perde (tam karartma)
- Honeycomb plise perde (yalıtım)
- Gece gündüz plise perde
- Özel ölçü üretim
- Türkiye geneli montajlı teslimat
- Avrupa'ya 4–10 iş günü içinde adrese teslimat
